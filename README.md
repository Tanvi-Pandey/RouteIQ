# RouteIQ 🚚

RouteIQ is a full-stack logistics and delivery optimization platform that demonstrates the practical application of core Data Structures and Algorithms (DSA) in solving real-world logistics and supply chain problems.

Built as a Data Structures & Algorithms capstone project, RouteIQ integrates efficient data structures with optimization algorithms to manage orders, prioritize deliveries, search inventory, and compute optimal delivery routes.

---

## Features

- 📦 Order Management
- 🚚 Delivery Route Optimization
- 📊 Analytics Dashboard
- 🔍 Fast Product Search
- ⚡ Priority-based Order Processing
- 🗺️ Graph-based Route Visualization
- 📈 Warehouse & Delivery Statistics
